Basket inflation

Baseline scope
Basket Inflation Since Start is defined as ((latest basket index ÷ baseline basket index) − 1) × 100. Positive percentages are shown in red (inflation), while negative percentages are shown in green (deflation).
How to read these charts: Basket Inflation shows basket cost over time in the selected unit. Basket cost is computed as the weighted average cost of the included basket items. Basket index is normalized to 100 at the baseline snapshot. Basket Change by Item uses USD delta per item (latest USD − baseline USD); all item deltas add up to the total basket USD move, where positive values push inflation higher and negative values pull it lower.
